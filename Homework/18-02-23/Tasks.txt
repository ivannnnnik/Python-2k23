Задание  №1
Написать программу, которая будет определять пройденное расстояние.
Программа принимает на вход скорость(v) и время(t).
Выводит пройденное расстояние.

Задание  №2
Написать программу, которая будет выводить наибольшее из двух чисел.
Принимает на вход x и y - оба числа типа 'int'.
Выводит наибольшее.

Задание  №3
Написать программу, которая будет определять диапазон числа.
Принимает на вход число x - тип 'int' .
Выводит диапазон числа по десяткам.
Пример: Input: 23
        Output: Диапазон числа: 20-30

Задание №5
Написать программу, которая будет определять диапазон числа.
Принимает на вход свое имя(name) и возраст(age).
Если возраст пользователя больше 18, то вывести строку
            "Добрый вечер, name! Вы совершеннолетний, поздравляем!",
Иначе вывести строку
            "Привет, name! Приносим извинения, но вы не можете гулять после 22:00"
